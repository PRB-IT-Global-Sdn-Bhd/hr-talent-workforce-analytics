# IT Human Resource — Talent & Workforce Analytics

> **PRB IT Global Sdn Bhd** | Custom Software for SME Engineering & Construction Clients (Malaysia)

## Overview

A **Talent Management and Workforce Analytics** platform that empowers engineering and construction SMEs to attract, develop, and retain skilled talent. The system provides deep insights into workforce composition, skills gaps, performance trends, and succession planning — enabling strategic human capital decisions.

## Key Features

| Module | Description |
|---|---|
| Talent Acquisition | Job posting, applicant tracking, and onboarding workflows |
| Skills Matrix | Competency mapping and CIDB Green Card tracking |
| Performance Management | KPI setting, appraisals, and 360-degree feedback |
| Learning & Development | Training records, certification tracking, and e-learning |
| Succession Planning | Talent pipeline and high-potential employee identification |
| Workforce Analytics | Headcount trends, attrition analysis, diversity metrics |
| Predictive Retention | ML models to predict and prevent employee turnover |

## Technology Stack

- **Frontend:** React.js, TypeScript, TailwindCSS, D3.js
- **Backend:** Python (FastAPI / Django)
- **Database:** PostgreSQL
- **AI/ML:** Python, scikit-learn, Pandas
- **Auth:** OAuth2 / RBAC
- **Hosting:** AWS / Azure (Malaysia region)

## Target Users

HR managers, talent acquisition specialists, department heads, and C-suite executives.

## Getting Started

```bash
git clone https://github.com/PRB-IT-Global-Sdn-Bhd/hr-talent-workforce-analytics.git
cd hr-talent-workforce-analytics
pip install -r requirements.txt
npm install
npm run dev
```
